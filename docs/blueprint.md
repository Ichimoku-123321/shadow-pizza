# **App Name**: Shadow Slice

## Core Features:

- Product Catalog: Displays standard products (pizza, drinks) with high-quality images.
- Product Selection Modal: A smooth modal appears when a user clicks on a product, showing detailed information and an 'Add to Cart' button.
- Custom Pizza Builder: Section where users can select a base and add ingredients, each ingredient affecting the total cost. This is implemented through simple item selection.
- Cart System: A slide-out cart that appears from the right side of the screen, allowing users to finalize order
- Navigation Menu: Header with a 'Golden Burger' style menu at the top right for browsing available categories.

## Style Guidelines:

- Background color: Light black (#1a1a1a) for a modern and luxurious feel.
- Primary color: Elegant gold (#D4AF37) used as a basis for the gradients, to provide an elegant style.
- Accent color: Mellow yellow (#FFDA63), also used in gradients with the primary color, to complement the modern style
- Headline font: 'Cinzel', a serif font for headings and titles.
- Body font: A sans-serif font (unspecified) for the main text content to improve readability.
- Subtle frame motion to transition colors from gold to yellow in interactive elements like buttons, in order to give a high quality impression.
- Modal appearance animations for a polished user experience.
